# PowerVision S1のアクティベートを通すproxyサーバー

今、安価に買えるPoweVision S1ですが、今からアクティベートを通そうとしても
アクティベートサーバーが稼働していないため出来ません。
今回のこのproxyはそれを簡易的に可能にしています。

## 使用方法

PCにpythonをインストールしておいてください。<br>
コマンドプロプトでcloneしたフォルダまで移動する。<br>
「python powervision_proxy.py」と入力して実行する。<br>


以下はスマホの設定になります。<br>
### iPhone

「設定」を開きます。<br>
「Wi-Fi」セクションに移動します。<br>
接続されているWi-Fiネットワークの横に「i」アイコンが表示されますので、それをクリックします。<br>
開いたページの下部に「HTTPプロキシ」セクションがありますので、「手動」オプションを選択します。<br>

- Server : <<pythonを実行するPCのIP>><br>
- Port : 8888<br>
- 保存をクリック<br>


#### Android
設定を開きます。<br>
Wi-Fiセクションに移動します。<br>
プロキシを使用して接続したいネットワークの名前を長押しします。<br>
「ネットワーク設定の編集/管理」をタップします。<br>
開いたダイアログボックスで、「詳細オプションを表示」をタップします。<br>
プロキシセクションのドロップダウンメニューをタップし、「手動」オプションを選択して手動で設定します。<br>

ホスト名: <<pythonを実行するPCのIP>><br>
ポート: 8888<br>
保存をクリック<br>

### 2. 「Vision+S1」を起動します。

アプリを起動し、「有効化」をクリックして任意の電話番号と認証コードを入力しれください。<br>
（入力中にネットワークエラーのPOPが出るかもしれませんが、無視して続行してください。）<br>

### 3. 重要 !!!!!!!!!! proxyを無効にする

#### iPhone
設定 > Wi-Fi > 「i」アイコン（Wi-Fiネットワーク名） > プロキシを設定 > 「オフ」を選択して保存をタップ

#### Android
設定 > Wi-Fi > Wi-Fi -> 「Wi-Fiネットワーク名」の(i)アイコン > プロキシ > なし
