# PowerVision S1のアクティベートを通すproxyサーバー

今、安価に買えるPoweVision S1ですが、今からアクティベートを通そうとしても
アクティベートサーバーが稼働していないため出来ません。
今回のこのproxyはそれを簡易的に可能にしています。

## 使用方法

PCにpythonをインストールしておいてください。
コマンドプロプトでcloneしたフォルダまで移動する。
「python powervision_proxy.py」と入力して実行する。


以下はスマホの設定になります。
### iPhone
「設定」を開きます。
「Wi-Fi」セクションに移動します。
接続されているWi-Fiネットワークの横に「i」アイコンが表示されますので、それをクリックします。
開いたページの下部に「HTTPプロキシ」セクションがありますので、「手動」オプションを選択します。

- Server : <<pythonを実行するPCのIP>>
- Port : 8888
- 保存をクリック


#### Android
設定を開きます。
Wi-Fiセクションに移動します。
プロキシを使用して接続したいネットワークの名前を長押しします。
「ネットワーク設定の編集/管理」をタップします。
開いたダイアログボックスで、「詳細オプションを表示」をタップします。
プロキシセクションのドロップダウンメニューをタップし、「手動」オプションを選択して手動で設定します。

ホスト名: <<pythonを実行するPCのIP>>
ポート: 8888
保存をクリック

### 2. 「Vision+S1」を起動します。

アプリを起動し、「有効化」をクリックして任意の電話番号と認証コードを入力しれください。
（入力中にネットワークエラーのPOPが出るかもしれませんが、無視して続行してください。）

### 3. 重要 !!!!!!!!!! proxyを無効にする

#### iPhone
設定 > Wi-Fi > 「i」アイコン（Wi-Fiネットワーク名） > プロキシを設定 > 「オフ」を選択して保存をタップ

#### Android
設定 > Wi-Fi > Wi-Fi -> 「Wi-Fiネットワーク名」の(i)アイコン > プロキシ > なし
